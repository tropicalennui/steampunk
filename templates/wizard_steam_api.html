SteamPunk needs your Steam API key to read your library. This key is stored only on this machine and is never shared.

Steam privacy requirements

Library sync works regardless of your privacy settings.

Achievement sync requires your Steam profile privacy settings to have both Profile and Game details set to Public.

You can change these at any time: Steam → Edit Profile → Privacy Settings.
